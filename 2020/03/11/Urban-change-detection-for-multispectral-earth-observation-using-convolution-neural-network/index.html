Urban change detection for multispectral earth observation using convolution neural network

2020-03-11 paper translation 0 评论

arXiv:1810.08468v1  [cs.CV]  19 Oct 2018

论文地址：https://arxiv.org/abs/1810.08468

 

ABSTRACT

本文探讨了使用CNN对多光谱遥感图像的变化检测。提出了像素级标注的变化检测数据集 OneraSatellite Change Detection (OSCD) dataset，还提出了两种可用于变化检测的监督学习的网络结构，并使用OSCD进行 trained from scratch 。

1. INTRODUCTION

1.1. RelatedWork 

由于缺少 labeled dataset，最近有一些基于迁移学习的方法被提出，但它们有一些限制：比如大多数CNN网络都是用RGB三通道的数据训练的，但是多光谱图像可能有13个通道，这就使得一些通道的信息没法充分利用。

此外，最近的一些方法更多的是手动地设计阈值，而不是端到端的训练。

1.2. Contributions

主要有两点：

1. (OSCD) dataset will be openly available on the internet

2. proposal of two different CNN architectures that aim to learn end-to-end change detection from this dataset in a fully supervised manner

2. DATASET

该数据集专注于城市区域，仅urban growth and changes被标记为 Change ，而像 vegetation growth 或 sea tides 的 natural changes 被标注为 no Change。

2.1. Challenges and limitations

由于卫星的分辨率较低，一些小的变化（如the appearance of small buildings,the extension of existing buildings or the addition of lanes to an existing road）不明显。

图像对间隔时间较短，图像包含标记为无变化的像素比标记为变化的像素多很多倍。

3. CHANGE DETECTION METHODS

仅在OSCD数据集中对其进行训练。与以前仅使用CNN的方法（ build difference images which are later thresholded）不同，我们进行端到端的训练。

以一个像素为中心的15*15*C （C为通道数）的patch，放入网络，最终做的是一个二分类的问题，即Change还是NoChange，作为这个patch中心像素的类别。

此外，还可以将二分类的阈值设置为0.5之外的值（ in case false positives or false negatives are more or less costly in a given application ）。

这比计算图像之间的简单差异要好得多，因为它涉及对更改的语义解释。

3.1. Architectures

（1）EarlyFusion(EF)，先把两个patch 拼接起来，网络的输入就变成了单个patch（15*15*2C）。然后，processed by a series of seven convolutional layers and two fully connected layers,where the last layer is a softmax layer with two outputs。

（2）Siamese (Siam) network，用孪生网络的方式，process each of the patches in parallel by two branches of four convolutional layers with shared weights, concatenating the outputs and using two fully connected layers to obtain two output values as before.

3.2. Full image change maps 

这种方式在预测阶段处理速度会很慢（因为以每个像素为中心的patch都要预测一次），作者提出可以在提取patch的时候使用更大的步长，然后用一个 voting system 来预测整张图象的像素。每个 classiﬁed patch 都会根据网络的输出和覆盖其中心像素的2D高斯分布设置权重，对其覆盖的所有像素的标签进行投票。也就是说，一个patch中，离中心像素越近，权重就会越大。

4. RESULTS

由于数据集不是很大，因此通过使用所有可能翻转的旋转来进行数据增强。

为了解决不同通道上的不同分辨率问题，将分辨率低于10m的通道上采样到10m分辨率，以便所有通道都可以以对齐的像素连接在一起。

标记为无变化的像素数比标记为变化的像素数多得多。 因此，在训练中，我们对 change class 使用更高的 penalization weight。

为了评估分类中输入通道数量的影响，比较了四种情况：彩色图像（RGB，3个通道），分辨率为10m的层（RGB +红外，4个通道），分辨率最高为20m的层（10 通道），分辨率最高可达60m的（13个通道）。

5. CONCLUSION

推动了下一步的工作，即尝试使用全卷积的网络为图像中的所有像素自动生成标签，从而减少所提出方法的 patch effect 。 将这项工作扩展到处理变化的 semantic labelling 也将是有意义的，这将进一步有助于解释image pairs。 最后，对以后能够处理用于变化检测的 image sequences 是有用的。
